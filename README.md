# Raining Characters

Raining Characters is a terminal-based procedural ASCII weather visualizer. It retrieves real-time conditions and coordinates from the free Open-Meteo API to generate infinitely scrolling, procedurally generated horizons with parallax-drifting cloud blocks.

## Quickstart
Ensure Python 3.10+ and [`uv`](https://docs.astral.sh/uv/) are installed. Then, run the application:

```bash
uv run main.py
```

## Controls

* **`Esc` or `q`**: Exit the weather visualizer.
